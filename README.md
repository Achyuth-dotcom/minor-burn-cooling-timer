# 🔥 BurnGuard – Minor Burn Cooling Timer

BurnGuard is a simple web application that helps users perform proper first aid for **minor burns** by guiding them through the **recommended 20-minute cooling period** with an easy-to-use countdown timer.

🌐 **Live Demo:** :contentReference[oaicite:0]{index=0}

---

## 📖 About

When someone suffers a minor burn, one of the most important first-aid steps is to cool the burn under **cool running water for 20 minutes**. BurnGuard provides a dedicated timer so users can accurately complete this recommended cooling period without guessing the time.

> **Note:** This application is intended for educational and first-aid assistance purposes only. It is **not** a substitute for professional medical advice or emergency care.

---

## ✨ Features

- ⏱️ 20-minute countdown timer
- ▶️ Start, Pause, and Reset controls
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and distraction-free interface
- ⚡ Lightweight and runs entirely in the browser
